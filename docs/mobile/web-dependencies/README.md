# Web Dependencies Configuration Documentation

This folder tracks all web-related dependency configurations, updates, and improvements.

## Purpose
Document every dependency change, version update, and configuration modification for:
- Web app dependencies (Next.js, React, etc.)
- Shared packages
- Build tools
- Development dependencies

## File Naming Convention
- `YYYY-MM-DD-dependency-name.md` - Specific dependency updates
- `tech-stack-improvement-YYYY-MM-DD.md` - General improvements
- `package-updates-YYYY-MM-DD.md` - Batch package updates

## Current Tech Stack
- Next.js 16 (App Router)
- React 19
- TypeScript 5 (strict mode)
- Tailwind CSS 4.1
- Supabase
- TanStack Query 5.90
- React Hook Form 7.66 + Zod 4.1

## How to Document Changes
1. Create a new dated file
2. Document what changed
3. Document why it changed
4. Document what broke (if anything)
5. Document how to revert (if needed)
