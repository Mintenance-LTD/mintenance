# Training Data Directory

This directory contains training data and templates for the Building Surveyor AI training pipeline.

## Files

- `ground-truth-labels-template.csv` - Template CSV file for ground truth labels
- `shadow-mode-predictions.csv` - Generated by export script (after running shadow mode batch)
- `shadow-mode-batch-*.json` - Summary reports from shadow mode batch execution

## Quick Start

1. **Create your ground truth CSV**: Copy `ground-truth-labels-template.csv` and fill in your labeled images
2. **Run shadow mode batch**: `npx tsx scripts/run-shadow-mode-batch.ts training-data/your-labels.csv`
3. **Export training data**: `npx tsx scripts/export-shadow-mode-data.ts`
4. **Train models**: Follow the steps in `docs/TRAINING_PIPELINE.md`

## CSV Format

See `docs/TRAINING_PIPELINE.md` for the complete CSV format specification.

Required columns:
- `filename` - Image filename
- `image_url` - Full URL to image
- `true_class` - Ground truth damage class
- `critical_hazard` - true/false

Optional columns:
- `property_type` - residential, commercial, rail, etc.
- `property_age` - Age in years
- `region` - Geographic region
